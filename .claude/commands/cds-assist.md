# /cds-assist

Delegates to the CDS-Assist agent for design system-compliant code and Figma generation.

## Usage

```
/cds-assist {your request}
```

## Examples

```
/cds-assist build a work orders dashboard
/cds-assist create a vendor registration form
/cds-assist build a permit detail page
```
